# Ash. — Personal Portfolio

A modern cinematic developer portfolio built with React and Tailwind CSS, featuring immersive UI animations, interactive layouts, responsive design, and smooth user experiences.

## 🚀 Live Demo

🔗 https://my-portfolio-git-main-adarshs-projects-a13293ae.vercel.app/

---

## ✨ Features

- Modern cinematic UI design
- Responsive layout for all devices
- Smooth scroll reveal animations
- Interactive project showcase
- Animated backgrounds and lighting effects
- Mobile-friendly navigation
- Clean and immersive user experience

---

## 🛠️ Tech Stack

- React
- Tailwind CSS
- JavaScript
- Framer Motion

---

## 📂 Project Setup

Clone the repository:

```bash
git clone https://github.com/adarshas-dev/portfolio.git
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```
